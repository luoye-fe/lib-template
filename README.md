library template
-----------------------

A simple js librart template.  

#### what inlude?

* es6/7

* rollup

* yarn

#### how to use?

* install `vue-cli` global.  

    `npm install vue-cli -g`

* use `vue-cli` to generate.

    `vue init luoye-fe/lib-template ./temp`

* install `node_modules`. (recommended to use `yarn`)
    
    `yarn install`

* development
    
    `yarn run dev`

* build
    
    `yarn build`
