library template
-----------------------

A simple js library template.  

#### what inlude?

* es6/7

* rollup

* eslint

#### how to use?

* install `vue-cli` global.  

    `npm install vue-cli -g`

* use `vue-cli` to generate.

    `vue init luoye-fe/lib-template ./temp`

* install `node_modules`. 
    
    `npm install`

* development
    
    `npm run dev`

* build
    
    `npm run build`
